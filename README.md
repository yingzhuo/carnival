# Carnival

#### 要求

* Java 8+
* SpringBoot 2.3.0+

#### 下载

* 推荐使用[maven](https://search.maven.org/search?q=carnival)

#### 安装最新的版本

```sh
git clone git@github.com:yingzhuo/carnival.git
mvn -f carnival/pom.xml clean install
```
